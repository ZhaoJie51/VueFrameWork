# vue-frame-work

## Project setup
```
npm install
```

### 命令
```
npm run serve             运行开发环境
npm run serve:test        运行测试环境
npm run serve:pro         运行生产环境
npm run build             打包开发环境
npm run build:test        打包测试环境
npm run build:pro         打包生产环境

npm run analyz            分析SPA应用
```

## 目录结构

```
├── dist                # 打包生成文件夹
│
├── node_modules        # 依赖包文件夹
│
├── pubilc              # 静态文件夹
│   ├── img             公共资源图片文件夹
│   ├── favicon.ico     ico图片
│   └── index.html      项目入口文件
│
├── src                 # 项目文件
│   ├── assets          静态文件夹
│   ├── compnents       公用组件文件夹
│   ├── config          环境配置文件夹
│   ├── mock            mock本地配置文件夹
│   ├── router          路由配置文件夹
│   ├── store           状态的保存文件夹
│   ├── utils           公用js文件夹
│   ├── views           子组件文件夹
│   ├── App.vue         主组件
│   └── main.js         入口JS文件
└── package.json        # 模块基本信息
```
